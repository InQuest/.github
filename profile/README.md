[OPSWAT](https://www.opswat.com) has acquired [InQuest](https://www.inquest.net), a specialized cybersecurity firm renowned for Deep File Inspection and RetroHunting technologies that protect critical DoD infrastructure. This merger combines InQuest's advanced threat intelligence and network detection capabilities with OPSWAT's MetaDefender platform, creating a unified defense system for federal and critical infrastructure organizations. The integration positions both companies to deliver more sophisticated prevention-based protection against evolving network threats while strengthening their competitive position in the federal cybersecurity market.
